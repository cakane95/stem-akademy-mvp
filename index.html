Explique-moi les boucles `for`.

Bien sûr ! Imagine un tailleur pour la Tabaski. Pour chaque client, il répète les mêmes actions : prendre les mesures et les noter dans son carnet. Une boucle `for`, c'est pareil :

POUR chaque client DANS liste_clients:
  PRENDRE les mesures
  NOTER dans le carnet

Comme tu le vois, une boucle `for` te permet de dire à un ordinateur de répéter une action pour chaque élément. Ça te permet aussi de réduire les lignes de code à écrire.
